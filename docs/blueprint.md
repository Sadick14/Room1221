# **App Name**: SRH Chatbot 1221

## Core Features:

- Confidential Chat Access: 24/7 access to SRH information via WhatsApp, Telegram, SMS, and a web app, with anonymous sessions.
- Privacy Features: Includes auto-delete history and a panic exit button.
- Multi-Lingual AI Support: AI chatbot capable of responding in English, Twi, Ewe, and Ga, through text and voice inputs.
- Safe Referral System: AI-powered referral system to match users with nearby verified youth-friendly clinics and pharmacies. The LLM powering this feature will act as a tool.
- Knowledge Enhancement: Use of story mode and myth busters to enhance knowledge.
- Personalized Chatbot: Users can rename the chatbot (e.g., 'Kofi,' 'Ama') for a more personal experience.

## Style Guidelines:

- Primary color: Soft green (#90EE90). Evokes a sense of health, nature, and tranquility, suitable for a health-related application focused on youth.
- Background color: Lightest green (#F0FFF0). Maintains the connection with the primary color but provides a subtle, unobtrusive background that won't distract users.
- Accent color: Pale yellow (#FAFAD2). This analogous color provides a gentle contrast, helping important elements stand out without being overwhelming.
- Font: 'PT Sans', a humanist sans-serif font for both headlines and body, offering a balance of modernity and warmth for a friendly user experience.
- Use clear, universal icons that are easily recognizable across different literacy levels.
- Maintain a clean, single-column layout to ensure information is easily accessible, especially for users on mobile devices.
- Use subtle animations for interactions, such as a progress indicator during AI processing, to enhance user engagement without being distracting.
